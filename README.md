# Calculadora Java Swing

Calculadora funcional desarrollada en Java usando Swing y Apache NetBeans.

## Funcionalidades

- Suma, resta, multiplicación y división
- Operaciones múltiples
- Números decimales
- Botón DEL y RESET
- Reutilización del resultado

## Tecnologías utilizadas

- Java
- Swing
- Apache NetBeans
- ArrayList
- ActionListener

## Conceptos trabajados

- Interfaces gráficas
- Eventos
- Bucles
- switch
- parseDouble
- Manipulación de Strings

## Funcionamiento

La calculadora almacena números y operadores en un `ArrayList<String>` y después recorre la lista para calcular el resultado mediante un `for` y un `switch`.
